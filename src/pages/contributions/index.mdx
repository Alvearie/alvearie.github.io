## Getting involved with Alvearie

Alvearie is open source and we encourage all to use, provide feedback, and contribute to the projects. The first step is to engage with the community. This can be done by:


  - **Engage with the community via our slack channels**<br/>
Log on to slack and join one or more slack channels at [alvearie.slack.com](http://alvearie.slack.com)

  - **Ask questions in our GitHub discussion boards**<br/>
Each of our GitHub repositories ([github.com/alvearie](https://github.com/alvearie)) has a GitHub Discussions board enabled where anyone can engage in a discussion, ask a question, or view discussion history.
