What is Project Alvearie?

The healthcare industry is evolving rapidly. Regulatory and cost pressures, interoperability requirements, and the changing delivery of care are creating an environment where developers are at the forefront of transformation.

Project Alvearie allows developers to leverage and build upon a collection of interoperable healthcare-specific data services and analytics tools developed by IBM, ecosystem partners, and the wider developer community. With those components, developers can build upon end-to-end healthcare data capabilities to solve for data access, ingestion, integration and analytics.

Better data flow means faster, smarter insights, which allows health enterprises to build innovative solutions that improve patient experience, quality of care, regulatory compliance, and IT costs. Health enterprises also benefit from an easy on-ramp to IBM professional services and capabilities.

Key Objectives / Features

  • Establish and grow an open ecosystem within the healthcare industry
  • Build and extensible, modular, scalable set of components supporting multiple cloud environments
  • Define reference implementations that highlight how to combine various technologies into meaningful patterns for healthcare
